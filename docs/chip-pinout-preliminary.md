## 2. ESP32 (U1)

> ESP32 является I2C-мастером; STM32 является slave (адрес 0x11).

### 2.1 Ethernet (KSZ8863RLL, RMII/SMI)

| GPIO  | Сигнал            | Примечание                     |
|-------|-------------------|--------------------------------|
| GPIO0 | EMAC_TX_CLK       | 50 МГц от PHY                  |
| GPIO4 | PHY_RST           | Сброс U4 KSZ8863RLL            |
| GPIO12| PHY_OSC_EN        | Включение тактового генератора  |
| GPIO19| EMAC_TXD0         | Ethernet TX data 0              |
| GPIO21| EMAC_TX_EN        | Ethernet TX enable              |
| GPIO22| EMAC_TXD1         | Ethernet TX data 1              |
| GPIO25| EMAC_RXD0         | Ethernet RX data 0              |
| GPIO26| EMAC_RXD1         | Ethernet RX data 1              |
| GPIO27| EMAC_RX_DV        | Ethernet RX data valid          |
| GPIO18| SMI_MDIO          | Management data I/O             |
| GPIO23| SMI_MDC           | Management data clock           |

> U4 = **KSZ8863RLL**. Интерфейс RMII/SMI подключен к ESP32, а Ethernet-линии дальше уходят дифференциальными парами на магнитные трансформаторы.

### 2.2 I2S аудио

| GPIO  | Сигнал       | Примечание                        |
|-------|--------------|-----------------------------------|
| GPIO2 | I2S0_SDIN    | I2S0 вход (кодек RX)              |
| GPIO5 | I2S0_SCLK    | I2S0 тактовый                     |
| GPIO15| I2S0_LRCK    | I2S0 LR-clock                     |
| GPIO32| I2S1_SCLK    | I2S1 тактовый                     |
| GPIO33| I2S1_LRCK    | I2S1 LR-clock                     |
| GPIO36| I2S1_SDOUT   | I2S1 выход (SENSOR_VP)            |
| GPIO39| SENSOR_VN    | Вход АЦП / I2S1 вторичный (уточнить назначение по схеме) |

> Линии I2S0/I2S1 используются аудиотрактом **ES8311 (U6)** / **ES7243 (U7)**; подробности по ролям кодека и АЦП приведены в §13.

### 2.3 I2C (к STM32 slave и аудиоустройствам)

| GPIO  | Сигнал   | Примечание                               |
|-------|----------|------------------------------------------|
| GPIO13| I2C_SDA  | I2C1 данные → STM32 PB7 (slave 0x11)    |
| GPIO14| I2C_SCL  | I2C1 тактовый → STM32 PB6               |

> Этот же мастер ESP32 конфигурирует аудиоустройства **ES8311 / ES7243** по I2C.

### 2.4 Прерывания и управление

| GPIO  | Сигнал       | Направление | Примечание                             |
|-------|--------------|-------------|----------------------------------------|
| GPIO34| STM32 INT    | INPUT       | Сигнал от STM32 (PB8 → ESP32)         |
| GPIO35| EXT_INT_ESP  | INPUT       | Внешнее прерывание для ESP32           |

### 2.5 Boot / UART0

| Сигнал | Назначение                                        |
|--------|---------------------------------------------------|
| U0TXD  | UART0 TX — программирование / консоль             |
| U0RXD  | UART0 RX — программирование / консоль             |
| IO0    | Вход режима загрузки (LOW = download mode)        |
| EN     | Аппаратный сброс ESP32 (также управляет U10/PN532)|

## 3. GPIO-экспандеры TCA6408A

### 3.1 U10 — выходы управления

| Pin | Сигнал       | Направление | Назначение                            |
|-----|--------------|-------------|---------------------------------------|
| P0  | STM32_RST    | OUTPUT      | Сброс STM32 (NRST)                    |
| P2  | LED_blue     | OUTPUT      | Синий светодиод статуса               |
| P3  | EN_1_12V     | OUTPUT      | Включение питания 12 В                |
| P5  | LED_red      | OUTPUT      | Красный светодиод статуса             |
| P6  | PA_EN        | OUTPUT      | Включение усилителя мощности          |
| P7  | STM32_BOOT0  | OUTPUT      | Управление BOOT0 для программирования |

### 3.2 U16 — входы мониторинга

| Pin | Сигнал        | Направление | Назначение                                  |
|-----|---------------|-------------|---------------------------------------------|
| P0  | ONE_SEC       | INPUT       | 1 Гц от DS3231M (SQW)                       |
| P1  | VBUS_DETECT   | INPUT       | Обнаружение USB VBUS                        |
| P2  | EXT_BTN_OPEN  | INPUT       | Внешняя кнопка открытия (через оптопару U18)|
| P3  | P70_IRQ       | INPUT       | IRQ от PN532 (уровень LOW)                  |
| P7  | EN (reset)    | OUTPUT      | Общая линия сброса: ESP32.EN + U10 + PN532  |

> **Примечание:** адрес U10 на шине I2C2 = 0x40. Адрес U16 предположительно 0x41 — требует проверки по схеме.

## 4. Системные цепи Reset / Boot

```
ESP32.EN ──┬──► U10.RESET
           ├──► PN532.~RSTPD
           └── управляется U16.P7

U10.P0  ──► STM32.NRST       (аппаратный сброс)
U10.P7  ──► STM32.BOOT0      (режим DFU/программирования)
```

### Разъём J2 (ESP32 download header)

| Pin | Сигнал | Назначение                        |
|-----|--------|-----------------------------------|
| 1   | NRST   | Сброс ESP32                       |
| 2   | BOOT0  | Режим загрузки                    |
| 3   | U0TXD  | UART0 TX для прошивки             |
| 4   | U0RXD  | UART0 RX для прошивки             |

### Зуммер (BZ1)

```
STM32.PA8 (TIM1_CH1) ──► Q1 (транзистор) ──► BZ1 (зуммер)
```

| Параметр | Значение  |
|----------|-----------|
| Таймер   | TIM1_CH1  |
| Длина 1  | 1000 мс   |
| Длина 2  | 20 мс     |

|----------|-----------|---------------------------------------------|

### Питание 12 В

| Сигнал   | Источник | Описание                              |

## 10. Внешняя кнопка открытия

```
J21 (EXT_BTN_OPEN) ──► U18 (оптопара) ──► U16.P2 (TCA6408A)
```

- Оптопара **U18** обеспечивает гальваническую развязку.
- Сигнал активный LOW.
- Дребезг обрабатывается программно (двухэтапный алгоритм, 200 мс подавление).

## 12. NFC — PN5321A3HN (U3, I2C2)

| Параметр | Значение        |
|----------|-----------------|
| Шина     | I2C2            |
| Линии    | I2C2: pin 27 = SCL, pin 28 = SDA |
| Адрес    | 0x48            |
| IRQ      | P70_IRQ (pin 25) → U16.P3 |
| ~RSTPD   | pin 38 ← общая цепь reset (ESP32.EN / U16.P7) |
| RF       | TX1/TX2/RX, контур 13.56 МГц |
| Питание  | VBAT, AVDD, TVDD, DVDD, PVDD |

> Имена выводов `NSS/P50_SCL` и `MOSI/SDA` — это альтернативные обозначения тех же выводов корпуса **PN5321A3HN**; в данной схеме они работают именно как линии I2C2 `SCL` / `SDA`.

## 13. Аудиоподсистема

### 13.1 ES8311 (U6)

| Интерфейс | Подключение | Примечание |
|-----------|-------------|------------|
| I2C       | → ESP32     | Управление кодеком со стороны ESP32 |
| I2S       | I2S0 / I2S1 | Раскладка линий берётся из §2.2      |

### 13.2 ES7243 (U7)

| Интерфейс | Подключение | Примечание |
|-----------|-------------|------------|
| I2C       | → ESP32     | Конфигурация АЦП со стороны ESP32    |
| I2S       | I2S0 / I2S1 | Использует раскладку линий из §2.2   |

### 13.3 Усилитель NS4150B (U9)

| Сигнал | Подключение | Примечание |
|--------|-------------|------------|
| ~SDB   | ← U10.P6    | `PA_EN`, включение моно усилителя |
| IN     | ← ES8311 OUTP/OUTN | Вход от аудиокодека |
