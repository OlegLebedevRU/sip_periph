# 📌 GM810 Краткая справка по диагностике и решению

## 🔍 Проблема была:

```
❌ gm810_restart_receive_it() НЕ вызывается
❌ service_gm810_uart_rx_callback() НЕ вызывается
```

## 🎯 Корневая причина:

**GPIO пины PA11/PA12 для USART6 не инициализировались в MX_GPIO_Init()**

Это означало, что:
- USART6 контроллер был готов ✓
- Но GPIO пины PA11/PA12 не были подключены к USART6_TX/RX функциям ✗
- Данные с GM810 не попадали в UART ✗
- Никаких прерываний не генерировалось ✗

## ✅ Решение:

**Добавлена GPIO инициализация в `Core/Src/main.c` (MX_GPIO_Init функция):**

```cpp
#if HW_PROFILE_GM810_USART6
  GPIO_InitStruct.Pin = GPIO_PIN_11 | GPIO_PIN_12;
  GPIO_InitStruct.Mode = GPIO_MODE_AF_PP;
  GPIO_InitStruct.Pull = GPIO_NOPULL;
  GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_VERY_HIGH;
  GPIO_InitStruct.Alternate = GPIO_AF8_USART6;
  HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);
#endif
```
