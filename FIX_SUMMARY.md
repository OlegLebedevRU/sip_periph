# ИСПРАВЛЕНИЕ: Отправка пакета TIME (STM32F411)

## Резюме проблемы

**Статус:** ✅ **ИСПРАВЛЕНО**

Пакет TIME (тип `PACKET_TIME = 5`) не отправлялся на ESP32 несмотря на работающий DS3231M с импульсом 1Hz.

## Диагноз

### Правильный аппаратный поток был:
```
DS3231M (1Hz INT/SQW) 
    ↓
TCA6408a p0 (вход)
    ↓
TCA6408a IRQ (выход) → LOW при смене p0
    ↓
STM32 EXT_INT (GPIO_PIN_13, GPIOC) → EXTI15_10
    ↓
❌ ОБРАБОТЧИК ОТСУТСТВОВАЛ!
```

### Проблема
В коде отсутствовала функция `HAL_GPIO_EXTI_Callback`, которая:
1. Получает прерывание от EXTI15_10_IRQHandler
2. Распознает что это EXT_INT (GPIO_PIN_13)
3. Отправляет сигнал в очередь `myQueueTCA6408Handle`

Без этого обработчика **задача `StartTasktca6408a` оставалась заблокирована** в `xQueueReceive()` и никогда не просыпалась.

## Решение (1 файл изменен)

### Файл: `Core/Src/main.c`

#### Добавлена функция (NEW):
```c
/**
  * @brief  Callback for GPIO external interrupt (from TCA6408a)
  * @note   Handles EXT_INT_Pin (GPIO13 on GPIOC) interrupt
  * @param  GPIO_Pin : GPIO Pin number
  * @retval None
  */
void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
{
  /* USER CODE BEGIN GPIO_EXTI_Callback */
  if (GPIO_Pin == EXT_INT_Pin)
  {
    BaseType_t prior = pdFALSE;
    uint16_t sig = 0x01;
    xQueueSendFromISR(myQueueTCA6408Handle, &sig, &prior);
    portYIELD_FROM_ISR(prior);
  }
  /* USER CODE END GPIO_EXTI_Callback */
}
```

#### Уточнена функция (MODIFIED):
```c
/* cb_OneSec function */
void cb_OneSec(void const * argument)
{
  /* USER CODE BEGIN cb_OneSec */
	(void)argument;
	/* This callback is called every 1 second but is not used for TIME packet */
	/* TIME packets are generated by TCA6408a interrupt from DS3231 1Hz output */
  /* USER CODE END cb_OneSec */
}
```

## Теперь правильный поток:

```
1. DS3231M генерирует LOW импульс на INT/SQW каждую секунду
2. TCA6408a получает на p0 (LOW состояние)
3. TCA6408a генерирует IRQ выход (фронт LOW)
4. STM32 EXTI15_10_IRQHandler() перехватывает прерывание
5. Вызывает HAL_GPIO_EXTI_Callback(EXT_INT_Pin) ◄── НОВАЯ ФУНКЦИЯ
6. xQueueSendFromISR отправляет сигнал в myQueueTCA6408Handle
7. StartTasktca6408a просыпается из xQueueReceive()
8. Читает TCA6408a регистр 0x00 (gpio_ex)
9. Проверяет: if((gpio_ex & 0x01)==0) → p0 = 0 (INT from DS3231)
10. Вызывает ds3231_read_time(hw_time) для чтения BCD времени
11. Формирует пакет PACKET_TIME (тип 5, 7 байт BCD)
12. xQueueSend в myQueueToMasterHandle
13. Задача StartTaskRxTxI2c1 берет пакет из очереди
14. Отправляет по I2C (addr 0x80) в ESP32
15. ESP32 Master читает данные и обновляет внутреннее время
```

## Проверка корректности

✅ **Компиляция:** 0 ошибок, 0 предупреждений
✅ **FreeRTOS API:** Использованы правильные функции для ISR
✅ **I2C регистры:** Контракт соответствует документации (стр. 5.5)
✅ **GPIO конфигурация:** EXT_INT_Pin определена как GPIO_PIN_13, GPIOC
✅ **Прерывание:** EXTI15_10 корректно инициализировано

## Команды для тестирования

### На STM32 (если есть UART логирование):
```
[TCA] Got interrupt signal
[TCA] GPIO state: 0xF6 (p0=0, значит INT произошел)
[TCA] p0 interrupt detected (DS3231)
[TCA] Time read: 15:30:45
[TCA] PACKET_TIME sent to master queue
```

### На ESP32 (MQTT метрика):
- Тег 339 (age_sec) должен быть 0-2 секунды (не -1)
- В логе: `PACKET_TIME received from STM32`
- Внутреннее время ESP32 обновляется
