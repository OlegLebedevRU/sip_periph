## 📋 Изменения в коде

### Файл: `Core/Src/main.c`

**Строка ~1801:** Добавлена новая функция
```c
void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
{
  if (GPIO_Pin == EXT_INT_Pin)
  {
    BaseType_t prior = pdFALSE;
    uint16_t sig = 0x01;
    xQueueSendFromISR(myQueueTCA6408Handle, &sig, &prior);
    portYIELD_FROM_ISR(prior);
  }
}
```

**Строки ~1760-1767:** Обновлена документация
```c
/* cb_OneSec function */
void cb_OneSec(void const * argument)
{
  (void)argument;
  /* This callback is called every 1 second but is not used for TIME packet */
  /* TIME packets are generated by TCA6408a interrupt from DS3231 1Hz output */
}
```

## 📡 Поведение после исправления

**Ожидаемое поведение:**
```
Каждую секунду:
1. DS3231M генерирует 1Hz импульс (обычный - не от моего кода)
2. STM32 получает прерывание EXTI15_10
3. HAL_GPIO_EXTI_Callback() вызывается (НОВОЕ)
4. Сигнал отправляется в очередь TCA6408
5. StartTasktca6408a просыпается
6. Читает время с DS3231
7. Отправляет PACKET_TIME в I2C Master
8. ESP32 обновляет свое внутреннее время
9. Процесс повторяется
```

**Логирование (если включено):**
```
[TIME] PACKET_TIME generated, age=0s
[TIME] ESP32: age_sec = 0, NTP valid, time synced
```
